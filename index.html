Часто задаваемые вопросы

Ответы на самые популярные вопросы об InGIR Pro

Какие системные требования у InGIR Pro?

Минимальные требования: Windows 10, Linux Ubuntu 20.04 или macOS 11, Python 3.8+, 4 GB RAM, 2 GB свободного места. Для 3D задач и больших сеток рекомендуется 8+ GB RAM.

Нужно ли устанавливать FEniCS отдельно?

Нет, InGIR Pro включает в себя все необходимые компоненты FEniCS. При первом запуске приложение автоматически проверит и установит недостающие зависимости.

Можно ли использовать InGIR Pro в коммерческих целях?

Бесплатная версия предназначена для образовательных и исследовательских целей. Для коммерческого использования требуется приобретение Pro или Enterprise лицензии.

Поддерживается ли работа на Linux без графического интерфейса?

Да, InGIR Pro может работать в режиме командной строки на Linux серверах. Для этого используйте опцию --headless при запуске.

Как часто выходят обновления?

Мы выпускаем обновления каждые 2-3 месяца с исправлениями ошибок и новыми функциями. Pro и Enterprise пользователи получают доступ к обновлениям раньше.

Есть ли документация на русском языке?

Да, InGIR Pro имеет полную документацию на русском языке, включающую руководство пользователя, API документацию и примеры решения типовых задач.
